🌳 Random Forest

Aprendizado com múltiplas árvores

🔍 slide com melhor explicação

🌲 O que é uma floresta?

  • Várias árvores de decisão
  • Cada uma aprende separadamente
  • Resultado final por votação
Aqui é onde entra o conceito principal. O Random Forest usa várias árvores. Cada uma aprende de forma independente. E no final, todas votam para decidir o resultado.

⚙️ Como funciona?

  1. Bootstrapping
  2. Criação das árvores
  3. Treinamento independente
  4. Votação
O modelo cria vários conjuntos de dados. Depois constrói várias árvores. Cada uma aprende sozinha. E no final todas participam da decisão.

🗳️ Classificação

  • Cada árvore faz uma previsão
  • As previsões são comparadas
  • A mais votada vence
Quando chega um novo dado, cada árvore dá um palpite. Depois o modelo conta os votos. E a classe mais votada é escolhida.

📊 Importância das Features

  • Identifica variáveis importantes
  • Baseado na separação dos dados
  • Ajuda na interpretação
O modelo também mostra quais variáveis são mais importantes. Ele analisa quais ajudam mais nas decisões. Isso ajuda a entender melhor o problema.

✅ Vantagens

  • Alta precisão
  • Menos overfitting
  • Robusto
O Random Forest é muito preciso. Ele evita overfitting. E funciona bem com muitos dados.

❌ Desvantagens

  • Mais lento
  • Mais pesado
  • Difícil de interpretar
Apesar de poderoso, ele pode ser mais lento. Também usa mais memória. E é mais difícil de interpretar.

🎤 Resumo

Várias árvores + votação = decisão mais confiável

Resumindo, o Random Forest combina várias árvores. E usa votação para tomar decisões melhores.
